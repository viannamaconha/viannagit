Passos para configurar o git

  1. Verificar as credenciais do Windows
  2. Configurar o nome e email do git
  3. Listar as configurações

Comandos:

verificar:
git config --list
configuração:
git config --global user.name "seu nome"
git config --global user.email "seu email"

Passos para o git monitorar uma pasta e salvar uma versão do projeto

  • criar uma pasta para o projeto no computador
  • entrar na pasta
  • git init
  • git add "nome do arquivo"
  • git commit -m "mensagem"

Passos para enviar o projeto salvo no computador para o Github

  • git branch -M Main
  • git remote add origin https://github.com/usuario/repositorio.git
  • git push -u origin main
